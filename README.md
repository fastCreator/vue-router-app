# vue-router-app
## 使用
```
npm install vue-router-app

import ElementUIExpand from 'vue-router-app'
Vue.use(ElementUIExpand)
```
## demo
https://fastcreator.github.io/vue-router-app/

## 发布pages以及npm
```
1. 修改packjson版本号
2. npm run deploy
```

## 开发
#### 运行
```
npm install
npm run serve
```
#### 新增组件
```
node add 组件名
```
